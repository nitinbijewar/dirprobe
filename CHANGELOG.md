# Changelog

## v1.0.0 (2026-03-21)

Initial release. Companion code for the CMS manuscript.

- Rank-2 directional diagnostics: D_dir, Delta_coh, S_align
- Temporal persistence: windowed A, block-to-full B
- Four-test robustness suite (gating, pooling, temporal, failure)
- 14 analytically controlled synthetic validation systems (A-N)
- Bundled KNN displacement data for 10 configurations
- Five reproduction modes: frozen, KNN end-to-end, generator
  end-to-end, combined, and smoke test
